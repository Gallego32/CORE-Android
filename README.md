# CORE-Android
Repositorio para el desarrollo de aplicaciones Android impartido por Core Dumped

### Hito 1
- **Primer hito sobre android en java.** Añadido el botón que podrá cambiar el texto de la parte inferior de la pantalla.
- Como tarea opcional he añadido una seekbar que modifica un valor y un texto.

### Hito 2
- **Segundo hito.** Añadida otra actividad para mostrar la información pedida al usuario en la actividad principal. Esta se compone de varias layouts, campos de texto, checkboxes y una seekbar.
- Como detalle observar el control sobre las checkboxes para evitar mantener varias activas notificándolo con un alert.
